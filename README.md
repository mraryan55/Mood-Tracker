# 😊 Mood Tracker App (Android)

A simple and user-friendly Android application that allows users to track their daily moods and emotions.

---

## 🚀 Features

- 😀 Select daily mood
- 📅 Track mood by date
- 💾 Save mood locally
- 📊 Simple and clean UI
- ⚡ Instant mood update

---

## 🛠️ Tech Stack

- Kotlin / Java
- Android Studio
- XML Layout
- SharedPreferences / Local Storage

---

## 📂 How It Works

1. User selects their current mood.
2. The selected mood is saved locally.
3. Mood can be viewed later by date.
4. UI updates instantly after selection.

---

This project was built to practice:

- User interaction handling
- Local data storage
- State management
- UI design principles
